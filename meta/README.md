# Meta assets

Images and audio used for **distribution** (Universal-DB / Universal Updater) and **CIA packaging** (3DS Home Menu). These files are not compiled into the `.3dsx` homebrew binary.

App logic lives in `source/`. Build output goes to `dist/`.

## Two pipelines

| Pipeline | When it matters | Key files |
|----------|-----------------|-----------|
| **Universal-DB** | Users browse/install/update via Universal Updater | `icon.png`, `meta/banner.png`, `meta/screenshot.png` |
| **CIA build** | Installed title on the 3DS Home Menu (top-screen banner, icon, startup sound) | `icon.png`, `meta/cia-banner.png`, `meta/audio.wav` |

UniStore and the Home Menu use **different files**. Updating `meta/banner.png` does not change an already-built CIA, and updating `meta/cia-banner.png` does not change the Universal-DB listing image.

## Regenerating banners

Source assets and a config-driven generator live in [`banner-src/`](banner-src/README.md).

| Command | Platform |
|---------|----------|
| `build.bat banners` | Windows (repo root) |
| `./build.sh banners` | Linux/Mac (repo root) |
| `meta/banner-src/build.bat` | Windows (direct) |
| `meta/banner-src/build.sh` | Linux/Mac (direct) |

**Release / version bump:** run one of the above (syncs `icon.png`, reads `VERSION`, writes `meta/banner*.png`). Then run `build_cia.bat` so the CIA embeds the fresh `meta/cia-banner.png`.

`make` and `build_cia.bat` **never** run the generator вЂ” they only consume files already in `meta/`.

Edit `banner-src/config.json` for layout changes; see [`banner-src/README.md`](banner-src/README.md).

## Editing guidelines

| Asset | Notes |
|-------|--------|
| `cia-banner.png` | Must stay **256Г—128**. Regenerate via `banner-src/`; layout is config-driven (do not hand-edit unless you must). |
| `audio.wav` | **16-bit PCM, 44100 Hz, stereo** for bannertool. Convert MP3 with ffmpeg if needed. |
| `banner.png` | Store listing; can differ from `cia-banner.png` (e.g. `outputs.cia-banner.title_lines` in config). |
| `icon.png` | Must stay **48Г—48** for 3DS icon format. Synced into `banner-src/assets/` by the banner build scripts. |

After changing CIA assets, run `build_cia.bat` and reinstall the new `.cia` to verify on hardware.
